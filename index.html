Sistema de Proteccion Web


Se desarrollo un sistema que protege los servidores de nuestro cliente que se especialza en la venta de productos quimicos, un sistema de seguridad robusto pero accesible y sencillo Dado a que nuestro cliente trabaja con informacion sensible y secreta que debe ser protegida es necesario que los protocolos de seguridad sean de alta exigencia.

Durante el proyecto se implementaron las mas recientes tecnologias como CASB para el intercambio de informacion entre los diferentes nodos de la red de la empresa.

Soluciones EDR Para satisfacer la necesidad de una protección continua contra amenazas en los ‘puntos finales’ (PC de escritorio, tablets, portátiles…) registrando en una base de datos centralizada eventos relativos tanto a los mismos como a la red, para a continuación analizar la base de datos en busca de factores que afecten a la seguridad del sistema.

31 de Junio, 2018 Lou Labs.Ltd

Optimizacion de Sistema de Habitaciones

En este trabajo optimizamos el sistema de una cadena de hoteles el sistema con el que contaban era un sistema antiguo que requeria una actulizacion para mejorar su rendimiento y porder mejorar el servicio del hotel

Utilizamos profilers para encontrar las secciones del programa que estaba tomando la mayor cantidad de recursos - el cuello de botella. Ocasionalmente se puede creer en dónde está el cuello de botella, pero esta intuición no siempre es correcta. La optimización de una pieza insignificante de código suele hacer poco para ayudar al rendimiento general.
